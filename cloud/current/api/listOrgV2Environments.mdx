---
title: "List of Environments"
api: "GET https://api.confluent.cloud/org/v2/environments"
contentType: "application/json"
description: "Retrieve a sorted, filtered, paginated list of all environments."
---

<Accordion title="Authorizations">

```
cloud-api-key
or
confluent-sts-access-token
```

<Icon icon="lock" /> **HTTP: cloud-api-key**
Authenticate with Cloud API Keys using HTTP Basic Auth. Treat the Cloud API Key ID as the username and Cloud API Key Secret as the password.

**HTTP Authorization Scheme**: `basic`

<Icon icon="lock" /> **HTTP: confluent-sts-access-token**
Authenticate with Confluent API using this credentials (JSON Web Tokens) following OAuth 2.0.

**Flow Type**: `clientCredentials` <br />
**Token URL**: `https://api.confluent.cloud/sts/v1/oauth2/token`
</Accordion>

## Query Parameters

<ParamField path="page_size" type="integer" default="10">
  A pagination size for collection requests.
</ParamField>

<ParamField path="page_token" type="string">
  An opaque pagination token for collection requests.
</ParamField>
